# LMODEM Notes

## Flow Control Codes
* DTR = Data Terminal Ready
* RTS = Ready to Send
* RTR = Ready to Receive

* ACK = Acknowledged
* NAK = Not Acknowledged

* EOF = End of File Blocks
* FIN = Fidle Transfer Complete
* CAN = Cancel file transfer operation 
* TOT = Time-Out Timer

## LMODEM LoStik Modes

### MODE1 - Fastest & Least Robust
* PWR = 6
* BW = 500
* SF = sf8
* CR = 4/6

### MODE2 - Middle Ground
* PWR = 12
* BW = 250
* SF = sf10
* CR = 4/7

### MODE3 - Slowest & Most Robust
* PWR = 17
* BW = 125
* SF = sf12
* CR = 4/8
